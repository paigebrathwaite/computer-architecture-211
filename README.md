This collection includes several C programming assignments focused on data structures, file processing, simulation, and problem solving. The projects range from smaller utility-style programs to more structured assignments involving matrix operations and logic circuit evaluation. Together, they reflect practice with pointers, dynamic memory allocation, recursion, parsing, linked data structures, and systems-oriented programming in C.

Assignment 1: C Data Structures and Utility Programs

This assignment includes a set of introductory C programs that each focus on a different core concept. The work includes string transformation, sorting, delimiter matching, linked lists, binary search trees, and matrix exponentiation. These projects helped build a foundation in memory management, recursion, file input, and classic data structure implementation.
	1.	rot13.c: Encodes text using the ROT13 substitution cipher by shifting alphabetic characters 13 places.
	2.	sorta.c: Sorts command-line input values.
	3.	balance.c: Checks whether parentheses, brackets, and braces are balanced using a stack-based approach.
	4.	list.c: Implements a sorted linked list with support for insertion, deletion, and printing.
	5.	bst.c: Implements a binary search tree with operations such as insert, search, delete, and traversal.
	6.	mexp.c: Computes matrix exponentiation from file-based input data.
	7.	Makefile: Build instructions for compiling each program.
	8.	tests1.txt and tests2.txt: Input files used for testing the balance checker.

Assignment 2: House Price Estimator

This project implements a house price estimator in C using matrix-based linear regression. It reads training data with known house features and prices, computes model weights through matrix operations, and uses those results to predict prices for new data. This project highlights file parsing, matrix multiplication, transposition, inversion, and practical numerical programming in C.
	1.	estimate.c: Reads training and test datasets, computes regression weights, and predicts house prices.
	2.	Makefile: Build instructions for compiling the estimator.

Assignment 4: Logic Circuit Truth Table Generator

This project generates truth tables for digital logic circuits described in an input file. It parses circuit definitions, evaluates each possible input combination, and prints the resulting outputs. The program supports common logic gates along with more advanced circuit components, making it a strong example of parsing and simulation in C.
	1.	truthtable.c: Parses circuit descriptions and generates a full truth table for the given logic design.
	2.	Makefile: Build instructions for compiling the truth table generator.
